# Biblioteca de Conhecimento

Este documento é um template de publicação para artistas compartilharem pesquisas, estudos de caso e aprendizados com a base interna do estúdio.

## Objetivo

- Registrar descobertas operacionais e criativas.
- Construir um repositório interno de boas práticas.
- Estimular a troca de insights entre artistas e gestores.

## Estrutura sugerida

### 1. Título do Estudo
> Ex: "O impacto da ficha de anamnese digital no atendimento premium"

### 2. Autor / Artista
> Nome, workspace e cargo.

### 3. Data

### 4. Resumo

- Breve visão do que foi analisado.
- Principais conclusões.

### 5. Contexto

- Qual era o problema ou desafio?
- Em que estúdio e em que fluxo isso ocorreu?

### 6. Metodologia

- Quais passos foram seguidos?
- Que ferramentas, formulários ou processos foram usados?

### 7. Resultados

- Dados quantitativos e qualitativos.
- O que melhorou? O que não funcionou?

### 8. Conclusões / Recomendação

- Recomendação para a equipe.
- Próximos passos.

### 9. Links / Anexos

- Referências internas.
- Modelos de formulário, planilhas ou imagens.

## Diretrizes para publicação

- Use linguagem clara e objetiva.
- Foque em ações práticas que outros artistas possam aplicar.
- Destaque lições aprendidas e evitar erros comuns.

## Benefícios para o estúdio

- Maior consistência operacional.
- Base de conhecimento viva e atualizada.
- Melhor alinhamento entre artista e administração.

> Este template é a base para transformar o CODEX em uma biblioteca real de conhecimento. Sempre que publicar uma nova pesquisa, o conteúdo deve ser compartilhado com o time para que o estúdio evolua junto.
